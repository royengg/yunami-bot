## ğŸ® How to Play

### 1. Create Your Profile
Before you can join an adventure, you need an identity.
```
/register
```
This will determine your base stats, personality, and starting inventory. Are you strong but reckless? Smart but fragile?

### 2. Gather Your Party
One person creates the lobby.
```
/party-create name:MidnightSquad
```

### 3. Invite Friends
Get your friends in. Most stories are best with **3 players**!
```
/party-invite user:@Friend
```

### 4. Choose Your Story
Start the adventure.
```
/startmultiplayer
```

### 5. Play!
- **Read the scene** on the main screen.
- **Check your DMs** for secret intel.
- **Click buttons** to make choices. (Only YOU can see choices for your role!)
